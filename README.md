# MOGRAPH STUDIO — Video Editor Portfolio

Premium cinematic portfolio built from **Google Stitch** design context.

## Design Source

- **Stitch Project:** Cinematic Motion Portfolio
- **Design System:** `.stitch/DESIGN.md`
- **Reference:** Uploaded JPG analyzed via Stitch (amber `#f89f43`, dark `#131313`, Plus Jakarta Sans + Hanken Grotesk + JetBrains Mono)

## Sections

1. Hero — headline, subheadline, CTAs
2. About Me
3. Featured Projects gallery (filterable)
4. Services
5. Skills
6. Testimonials
7. Contact — form + social links
